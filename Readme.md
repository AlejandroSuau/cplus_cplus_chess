# Chess Game - C++ and SDL

<img src="./demo/starting.png" width="350"> <img src="./demo/check.png" width="350">

## Technology
Compilation:
* C++20, CMake & Make.

Framework and libraries:
* SDL2
* SDL2_image
* SDL2_mixer
* SDL2_ttf
* pugixml

### Windows - Build and execute

```
cmake -B build`
cd build
cmake --build .
.\Debug\ChessGame.exe
```

### MacOS - Build and execute

```
cmake -B build
cd build
make
./ChessGame
```
